# CS50x Scratch Game

A space-themed interactive game developed in Scratch as part of Harvard CS50x.

## Overview

This project introduces core programming concepts including:

* Event-driven programming
* Variables and game logic
* User interaction
* Basic animation and game mechanics

## Features

* Interactive gameplay
* Space-themed environment
* Beginner-friendly implementation

## Technologies

* Scratch
